# Branding a ikony

Verze 1.8.2 zviditelňuje logo přímo v aplikaci a přidává cache-busting pro favicon a PWA ikony.

## Viditelné umístění

- hlavní administrace `/`
- landing page `/sales`, `/landing`, `/cs`
- veřejná rezervace `/booking` a `/booking/<slug>`
- onboarding `/onboarding`
- správa klientů `/admin/clients`

## Soubory

```text
public/assets/prague-ai-voice-logo.png
public/assets/prague-ai-voice-icon-512.png
public/assets/prague-ai-voice-icon-192.png
public/assets/favicon.png
public/site.webmanifest
```

Pokud prohlížeč stále ukazuje starou ikonu, otevřete stránku v anonymním okně nebo použijte tvrdé obnovení `Ctrl + F5`.
